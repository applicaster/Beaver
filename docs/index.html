Beaver

A macOS app for inspecting and controlling Applicaster's mobile SDKs over WebSocket. Live log feed, storages browser, command bar — and it stays up to date automatically.

Download for macOS

Universal binary · macOS 14 or later · release notes

Live log feed

Stream events from any connected client. Filter, exclude, search, highlight, jump to time, bookmark, and export — by JSON or filtered.

Storages browser

Inspect Session, Local, and Keychain storage. Expand namespaces, peek at long values, add or remove keys directly from the desktop.

Command bar

Send any SDK console command to the connected device. Discoverable catalog of commands with inline syntax help.

Auto-updating

Sparkle-powered updates land in-place. New releases are Ed25519-signed by CI; only Applicaster-built binaries are accepted.

Install

  1. Click Download for macOS. You'll get Beaver.zip.
  2. Unzip and drag Beaver.app into /Applications.
  3. Launch it. macOS may prompt once — the app is notarized by Apple, so a single approval is all it takes.
  4. From the second launch onward, updates arrive automatically via Sparkle. There's also Beaver → Check for Updates… in the menu bar.
